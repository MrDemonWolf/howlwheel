# Howlwheel — moved

> **This project now lives inside [Wolfathon](https://github.com/MrDemonWolf/wolfathon).**

Howlwheel (a wheel-of-dares spinner for Twitch streams) was merged into the
Wolfathon repo so there's **one repo, one stack, one deploy**. All future work,
issues, and releases happen there.

- Code + docs: **https://github.com/MrDemonWolf/wolfathon**
- The wheel is a dashboard tab (`/dashboard/wheel`) + an OBS overlay
  (`/overlay/wheel`), reusing Wolfathon's Cloudflare Access, overlay token, and
  brand UI.
